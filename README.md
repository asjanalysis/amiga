# Amiga Starfire

A colorful, browser-based, pixel-art sci-fi scrolling arcade game inspired by late-1980s home computer shooters and the visual energy of the Amiga 500 era.

## Play

Open `index.html` in a browser, or publish the repository with GitHub Pages.

## Controls

- Move: Arrow keys or WASD
- Fire: Space, Z, or X
- Pause: P
- Mute: M
- Touch/mobile: tap or drag on the game canvas

## Features

- Low-resolution pixel canvas scaled up with crisp edges
- Bright sci-fi palette with parallax stars, planets, comets, scanlines, and chunky sprites
- Multiple enemy types, escalating waves, boss-style carrier encounters, power-ups, score, combo meter, lives, shields, and local high score
- Self-contained static site with no build step and no external dependencies

## GitHub Pages

To publish:

1. Open the repository settings on GitHub.
2. Go to **Pages**.
3. Set source to **Deploy from a branch**.
4. Choose `main` and `/root`.
5. Save.

GitHub will provide the published URL after deployment.
